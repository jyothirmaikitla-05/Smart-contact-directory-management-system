# Smart Contact Directory Management System

A responsive web application developed using HTML, CSS, and JavaScript to efficiently manage contact information. The system allows users to add, view, search, and organize contacts through an intuitive and user-friendly interface.

## Features
- Add new contacts
- View all saved contacts
- Search contacts
- User-friendly interface
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
- `index.html` – Home page
- `about.html` – About the project
- `add-contact.html` – Add new contacts
- `view-contacts.html` – Display saved contacts
- `contact.html` – Contact page
- `style.css` – Styling
- `script.js` – Functionality
